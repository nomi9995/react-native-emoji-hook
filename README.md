# react-native-emoji-hook

write emoji by emoji code

## Installation

```sh
npm install react-native-emoji-hook
```

## Usage

```js
import EmojiHook from "react-native-emoji-hook";

// ...

const result = await EmojiHook.multiply(3, 7);
```
